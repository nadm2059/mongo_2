
# Task Manager

A simple full-stack to-do app built with Node.js, Express, MongoDB, and vanilla JavaScript.

## Features
- Add tasks
- View tasks
- Delete tasks

## Running Locally
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the server: `node server.js`

## Technologies
- Node.js
- Express.js
- MongoDB
- JavaScript (Frontend)
